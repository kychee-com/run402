Deploy a full webapp in 2 calls

# 1) Provision a backend (database + REST + auth + storage)
POST /v1/projects  [x402: testnet, free]
# Returns: project_id, anon_key, service_key

# 2) Create tables, set up auth, add data...
POST /admin/v1/projects/:id/sql
"CREATE TABLE todos (id serial primary key, task text, done boolean);"

# 3) Deploy the frontend
POST /v1/deployments  [x402: $0.05]
{
  "name": "my-todo-app",
  "files": [
    { "file": "index.html", "data": "<!doctype html>..." },
    { "file": "style.css",  "data": "body { ... }" },
    { "file": "app.js",    "data": "fetch('/rest/v1/todos')..." }
  ]
}
# Returns: https://my-todo-app.sites.run402.com — live now

# Total cost: $0 real money (testnet). Total time: seconds.
