From zero to live app in seconds

# 1) Get free testnet USDC (no wallet needed)
POST /v1/faucet
# Returns: wallet address, private key, 1.00 USDC on Base Sepolia

# 2) Provision a database (one call, one payment)
POST /v1/projects  [x402: $0.10]
# Returns: project_id, anon_key, service_key — ready immediately

# 3) Create tables
POST /admin/v1/projects/:id/sql
"CREATE TABLE todos (id serial primary key, task text, done boolean);"

# 4) Use the REST API
GET  /rest/v1/todos?done=eq.false
POST /rest/v1/todos  {"task": "Ship it", "done": false}

# 5) Deploy a frontend
POST /v1/deployments  [x402: $0.05]
# Returns: https://<id>.sites.run402.com — live URL
