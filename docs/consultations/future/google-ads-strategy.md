## 7) Tracking: this is the hard part, and it matters a lot

Because your conversion may happen **outside the browser** after the user goes back to their agent, normal GA4 tracking will undercount badly.

### I would track these conversions
Primary:
- **first funded allowance**
- **first successful app provision**
- **first Hobby/Team deployment**
- **first fork**
- **renewal / second project**

Secondary:
- copied starter prompt
- clicked docs
- viewed pricing
- started testnet flow
- quote requested

### Important implementation detail
Capture **GCLID** from the ad click and carry it into the real action.

Best way:
- when user clicks **Copy starter prompt**, generate a unique referral token
- include that token in the copied prompt or a personalized `llms.txt` URL
- when the agent later provisions an app, tie it back to the original ad click
- import that as an **offline conversion** into Google Ads

If you don’t do this, Google will optimize toward vanity actions like pageviews and prompt copies.

## 8) Bidding / targeting settings

I’d start with:

- **Search only**
- **turn off Display expansion**
- **turn off Search Partners initially**
- **US only** at first, then expand
- **English only**
- **desktop-heavy** bias
- **exact + phrase match** first
- low-budget DSA only for mining long-tail queries from your use-case pages

### Bidding
- Start with **Max Clicks** or conservative CPC control while gathering data
- switch to **Max Conversions** once you have enough imported real conversions
- later, move to value-based bidding if you can send:
  - allowance amount
  - project tier
  - renewal/fork value

## 10) What I would NOT do

- **Don’t** buy broad generic keywords like:
  - web hosting
  - postgres hosting
  - app builder
  - website builder
  - cloud hosting
- **Don’t** run Performance Max early
- **Don’t** send paid traffic to the homepage by default
- **Don’t** optimize around “free” unless you’re comfortable with low-intent traffic
- **Don’t** lead with USDC/x402 on mainstream campaigns
- **Don’t** judge success by the $0.10 prototype alone

That last point matters a lot: Google CPCs can easily exceed your first transaction.  
Paid search only works if acquired users become:
- repeat deployers
- allowance funders
- Hobby/Team users
- publishers with downstream forks

So optimize to **first meaningful deployment or funded allowance**, not just the first $0.10 event.
