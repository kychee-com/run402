## Our Pitch (Summary)

Run402 is a live implementation of x402. Agents autonomously read our llms.txt/MCP, generate a wallet, get testnet USDC from our faucet, pay via x402, and deploy a full app in ~3 minutes. No humans, no API keys, no email. Identity is just a wallet address.

**The gap:** No way for agents to autonomously onboard into a persistent wallet (Coinbase wallet needs human + email + OTP). Agentic Wallets and/or Stripe Issuing could be the missing piece.

**Demo:** Ask any coding agent to build something using run402.com/llms.txt
